# Pólya's How to Solve It — Cheatsheet

A one-page, double-sided, print-ready visual reference for George Pólya's problem-solving framework from *How to Solve It*.

## How to run

Open `index.html` in a browser.

## What it shows

Pólya's four-phase heuristic for problem-solving, originally described for mathematics but widely applicable:

1. **Understand the Problem**
2. **Devise a Plan**
3. **Carry Out the Plan**
4. **Look Back**

Also includes a list of the main heuristics from the book (analogy, auxiliary problems, specialization/generalization, usw.).
